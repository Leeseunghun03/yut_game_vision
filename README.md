# yut_game_vision
This package is a vision module for a robot Yut game on ROS2.
This package was used for the autonomous board game robot project.

## Env

1. Ros2 humble
2. Two Realsense camera
3. aruco marker

## Feature
1. Detect and Extract game map, game piece.
